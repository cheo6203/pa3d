3D 파일 활용 안내 ▼

이 페이지에서 다운로드하는 .glb 파일은 포털(pa.go.kr) 원본 그대로입니다. Blender 4.2로 포토그래메트리 방식 스캔 제작, PBR 재질 내장, 단일 파일 자기완결 구조입니다. 공공누리 제4유형(출처표시·변경금지)으로 제공되므로 출처 표시 후 원형 그대로 활용 가능합니다.

🌐 웹·모바일 서비스 삽입 입문
HTML 한 줄로 3D 뷰어를 웹페이지에 넣을 수 있습니다.
✓ Google <model-viewer> 태그 사용 시 AR(증강현실)까지 자동 지원
✓ iOS Safari → AR Quick Look, Android → Scene Viewer로 실물 크기 체험 가능
△ 다기세트(PJE0000234)는 텍스처 없어 색상이 다르게 표현됨
<model-viewer> Three.js Babylon.js WebXR A-Frame
🖨 3D 프린트 중급
GLB를 직접 슬라이서에 넣을 수 없어 STL/3MF 변환이 필요합니다.
✓ Blender에서 GLB 열기 → File › Export › STL로 변환
✓ 기념정·피나투보 조각상은 단일 메시라 변환 후 바로 프린트 시도 가능
△ 다기세트는 22개 분리 파츠 — 변환 후 슬라이서에서 파츠별 배치 필요
△ 포토그래메트리 스캔본이라 내부가 비어있지 않을 수 있음 (Meshmixer에서 watertight 보정 권장)
Blender (무료) Meshmixer (무료) PrusaSlicer Cura
🎮 게임엔진 · 메타버스 중급
GLB는 Unity·Unreal·Sketchfab이 모두 직접 임포트 지원하는 표준 포맷입니다.
✓ Unity: Package Manager에서 glTF 패키지 설치 후 Assets에 드래그
✓ Unreal Engine 5: glTF Importer 플러그인 활성화 후 임포트
✓ Sketchfab, Mozilla Hubs, Spatial.io 등 메타버스 플랫폼 직접 업로드 가능
△ 정점 수가 10~15만 개 수준으로 실시간 게임에 쓰려면 LOD 축소 권장
Unity Unreal Engine 5 Sketchfab Spatial.io Mozilla Hubs
🎬 교육 · 영상 콘텐츠 입문
Blender에서 GLB를 열어 360° 턴테이블 영상, 단면 컷어웨이 애니메이션 등을 제작할 수 있습니다.
✓ 수업자료·전시 보조영상으로 유물 실물을 대체 활용 가능
✓ After Effects, Premiere에 Blender 렌더 영상 합성
✓ PowerPoint 3D 개체 삽입(Windows) — GLB 직접 지원
Blender (무료) PowerPoint (Windows) After Effects DaVinci Resolve
🔬 디지털 보존 · 학술 연구 전문
포토그래메트리 스캔 메시를 학술 목적으로 분석·측정·변환할 수 있습니다.
✓ CloudCompare로 포인트클라우드 변환 및 치수 측정 가능
✓ IIIF 3D Working Group 표준 연계 — 국제 디지털 아카이브와 상호운용 기반
✓ MeshLab으로 메시 품질 분석, Hausdorff 거리 측정
△ 실측 스케일 메타데이터 없음 — 치수 계측 전 실물 참조 보정 필요
CloudCompare (무료) MeshLab (무료) Blender IIIF 3D GIS 연계
🤖 AI · 생성 모델 활용 전문
3D 스캔 메시를 AI 학습·생성 파이프라인의 입력 데이터로 사용할 수 있습니다.
✓ NeRF(Neural Radiance Field) 재학습 입력 — 텍스처 내장 GLB는 멀티뷰 렌더로 활용 가능
✓ 3D 생성 모델(Shap-E, Point-E 등) 파인튜닝 소재
✓ Stable Diffusion ControlNet Depth 입력으로 텍스처 재생성
△ 공공누리 제4유형 — AI 생성물 배포 시에도 원본 출처 표시 필요
Blender + NeRF Studio Shap-E ControlNet Instant-NGP
📋 라이선스 필수 확인
모든 파일은 공공누리 제4유형(출처표시·변경금지)입니다. 출처(대통령기록관, pa.go.kr)를 반드시 표시하고, 파일 원형을 변경 없이 사용해야 합니다. 2차 가공(색상 변경, 메시 편집 등) 후 배포는 허용되지 않습니다. 상업적 이용 전 대통령기록관 담당부서 문의를 권장합니다.
